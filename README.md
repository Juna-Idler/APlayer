# APlayer

特定のフォルダにある音声ファイルを連続再生することを目的としたアプリケーションです。
おまけで画像ファイルやテキストファイルも表示できます。

マウスでも操作できますが、ゲームパッドを使って操作することを目的に作られています。

ただしリストの編集や設定の操作はわざわざゲームパッドでやるのは面倒なので、マウスのみの対応となります。
